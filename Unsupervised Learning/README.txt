The first 5 folders contain code and csv files for experiments 1 through 5. The last folder called util contain code for data analysis and other supporting documents.

All work was done in python, therefore to run, simple navigate to a the folder in
which a desired experiment was performed and in most cases, each folder contain two python scripts for our cancer dataset and our diabetes dataset.

Finally, run python3 filename.py
