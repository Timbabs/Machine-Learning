

#Randomized optimization

The code is run with ABAGAIL (https://github.com/pushkar/ABAGAIL), a library containing machine learning and artificial intelligence algorithms. AGAGAIL.jar has been included here for conveneince else use ant to otain the jar file.

To run, do the following

- python dataSplitter.py to split the data into train and test datasets
- javac -cp ABAGAIL.jar:. *.java
- java -cp AGABGAIL.jar:. NeuralNetsDriver
     *NOTE: Change paths in all files*
